# Project-tracker-UI

A multi-view project management UI build using react + typescript

------
# Live Demo : matheshwari-project-tracker-live.netlify.app

# Features : 

## Kanbanboard : 
      -drag and drop task between columns
      -custom drag and drop(no libraries)
## Listview :
      - sortable table
      -inline status change
      -virtual scrolling
-------
# Tech stack : 
      -React
      -Typescript
      -vite

----------

### installation :
      -git clone
      -cd project-tracker-ui
      -npm install
      -npm run dev

### key implementation :

- Drag and drop : Used native drag events for moving tasks between column.
- virtual scrolling : Rendered only visible rows using scroll position for better performance.
